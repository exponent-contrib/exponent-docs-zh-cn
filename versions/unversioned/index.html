了解 Exponent¶

Exponent是一个手机应用开发工具，可以帮助你用Javascript来开发原生IOS和Android应用，并且轻松部署应用。

写Exponent应用就好比写React Native应用，只是你再不需要用Xcode / Android Studio, 也不需要写任何native代码。Exponent SDK 提供了很多系统功能，比如联系人，摄像头还有社交登录。我们会持续添加更多功能到我们的SDK, 如果有些功能我们目前没有提供的，请告诉我们: support@getexponent.com.

Exponent也提供了各种用途的UI组件, 基本上所有应用都会用但是React Native核心不包含的。包括icons, image lightbox, loading overlay还有更多.

有时间的话不如玩一下. 先 安装 然后 up-and-running, 去写你的第一个Exponent应用吧!

如果你不是很熟悉React还有React Native, 你可以从 React Native Express 开始.
