SDK API Reference¶

Exponent SDK 提供一些系统功能，比如联系人，相机，社交登录等。npm 包: exponent. 可以通过在项目根目录 运行 npm install --save exponent 来安装包。然后你可以如下在你的 Javascript 代码里 import SDK 库:

import { Contacts } from 'exponent';

You can also import all Exponent SDK modules:

import * as Exponent from 'exponent';

然后你就可以写比如: Exponent.Contacts.getContactsAsync().
