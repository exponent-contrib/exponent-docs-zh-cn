exp.json 配置
========

.. This file is automatically generated! Do not edit it directly -- see scripts/generate-exp-docs.js

``exp.json`` 包含你 app 里除了代码以外的配置。下面是所有支持的属性:


.. attribute:: name

 **Required**. Exponent 客户端里 app 的名字，还有 standalone app 主屏的名字。

.. attribute:: description

 简短描述下你的 app 是什么， 为什么会让人觉得很棒。

.. attribute:: slug

 **Required**. 发布时候友好的 url name, 比如: ``exp.host/@your-username/slug``.

.. attribute:: sdkVersion

 **Required**. App 用到的 Exponent sdkVersion。应该和 package.json 里定义的版本同步。

.. attribute:: version

 你的 app 版本，你可以用任何喜欢的版本号规范。

.. attribute:: orientation
 锁定app, ``portrait`` 或者 ``landscape``。默认不锁定，支持 default, portrait, landscape。

.. attribute:: primaryColor
 在 Android 上，代表多任务时 app 的颜色。当前不支持 iOS, 但是以后可能会用到。
 16进制颜色代码, 比如: ``'#000000'``

.. attribute:: iconUrl

 App 的图标链接。我们推荐你用一张透明的 512x512 png。会显示在主屏，还有 Exponent 客户端里。
 
.. attribute:: notification

 推送通知配置。

   .. attribute:: iconUrl

    推送通知图标链接。48x48 png, 灰阶透明。

   .. attribute:: color

    推送通知图标显示在通知中心时候的 Tint color. 16 进制颜色代码，比如: ``'#000000'``

   .. attribute:: androidMode

    单独显示每次通知 (``default``), 或者折叠成一个 (``collapse``).
    默认 collapse。

   .. attribute:: androidCollapsedTitle

    如果 ``androidMode`` 设置为 ``collapse`` 的话, 用来显示折叠信息的标题。 比如: ``'#{unread_notifications} new interactions'``.

.. attribute:: loading

 当用户打开 app 时的加载 (拉取，缓存 Javascript bundle 还有 assets) 页面配置。
 
   .. attribute:: iconUrl

    启动 app 时候显示的图标。大小比例不限制，必须是 .png。
    Url that points to the icon to display while starting up the app. Image size and aspect ratio are up to you. Must be a .png.

   .. attribute:: exponentIconColor

    如果没有提供 icon 的话，我们会默认显示 Exponent logo。你可以选择 ``white`` 或者 ``blue``.

   .. attribute:: exponentIconGrayscale

    类似 ``exponentIconColor``, 但是指明灰阶 (``1``) 或者不是灰阶 (``0``)。

   .. attribute:: backgroundImageUrl

    加载页面背景图片。大小比例不限制，必须是 .png。      
    
   .. attribute:: backgroundColor

    加载页面背景颜色。
    16 进制颜色代码，比如: ``'#000000'``

   .. attribute:: hideExponentText

    Exponent 默认在加载页面末端显示一些文字。设置为 ``true`` 可以不显示。

.. attribute:: appKey
 Exponent 默认选择用 AppRegistry 注册的 ``main`` application。你可以修改这个属性如果需要更改的话。

.. attribute:: androidStatusBarColor

  16 进制颜色代码，比如: ``'#000000'``

.. attribute:: androidHideExponentNotificationInShellApp
 Exponent 默认为你的 app 添加一个通知，包含刷新按钮还有 debug 信息。设为 ``true`` 来关闭通知。

.. attribute:: scheme

 **Standalone Apps Only**. 廉价到你的 app 的 url scheme。比如， 如果设为 ``'rnplay'`` 的话, 点击 rnplay:// 链接就会打开你的 app。

.. attribute:: entryPoint

 你的 main Javascript 的相对路径。

.. attribute:: extra

 你想传给你的 experience 的额外参数。

.. attribute:: rnCliPath

 
.. attribute:: packagerOpts

 
.. attribute:: ignoreNodeModulesValidation

 
.. attribute:: nodeModulesPath

 
.. attribute:: ios

 **Standalone Apps Only**. iOS standalone app 相关配置

   .. attribute:: bundleIdentifier

    你的 iOS standalone app 的 bundle identifier。必须在 App Store 上保证唯一。可以看 `这个 StackOverflow question <http://stackoverflow.com/questions/11347470/what-does-bundle-identifier-mean-in-the-ios-project>`_.

    iOS bundle identifier 的命名，比如: host.exp.exponent, exp.host 是我们的域，Exponent 是我们的一个 app。

   .. attribute:: buildNumber

    你的 iOS standalone app 的 Build number。

   .. attribute:: config

    
       .. attribute:: fabric

        `Twitter Fabric <https://get.fabric.io/>`_ keys, 来支持 Crashlytics 和其他服务。

           .. attribute:: apiKey

            你的 Fabric API key

           .. attribute:: buildSecret

            你的 Fabric build secret

       .. attribute:: googleSignIn

        `Google Sign-In iOS SDK <https://developers.google.com/identity/sign-in/ios/start-integrating>`_ keys, standalone app 需要的.

           .. attribute:: reservedClientId

            保留的 client id url scheme。 可以在 `GoogeService-Info.plist` 里找到。

.. attribute:: android

 **Standalone Apps Only**. Android standalone app 相关配置。

   .. attribute:: package

    你的 Android standalone app package name。必须在 Play Store 上保证唯一。可以看: `这个 StackOverflow question <http://stackoverflow.com/questions/6273892/android-package-name-convention>`_.
    Reverse DNS notation unique name for your app. For example, host.exp.exponent, where exp.host is our domain and Exponent is our app.

    package name 是一个倒置 DNS 标记, 比如: host.exp.exponent, exp.host 是我们的域，Exponent 是我们的一个 app。
                  
   .. attribute:: versionCode

    Google Play 需要的版本号。每次发布递增 1, https://developer.android.com/studio/publish/versioning.html.
    
   .. attribute:: config

    
       .. attribute:: fabric

        `Twitter Fabric <https://get.fabric.io/>`_ keys, 来支持 Crashlytics 和其他服务。

           .. attribute:: apiKey

            你的 Fabric API key

           .. attribute:: buildSecret

            你的 Fabric build secret

       .. attribute:: googleMaps

        `Google Maps Android SDK <https://developers.google.com/maps/documentation/android-api/signup>`_ key, standalone app 需要。

           .. attribute:: apiKey

            你的 Google Maps Android SDK API key。

       .. attribute:: googleSignIn

        `Google Sign-In Android SDK <https://developers.google.com/identity/sign-in/android/start-integrating>`_ keys, standalone app 需要。

           .. attribute:: apiKey

            Android API key. 可以在 developer console 的 credentials 里或者 `google-services.json` 找到。

           .. attribute:: certificateHash

            用来 build apk 的签名证书的 SHA-1 hash, 不能包含 `:`, 
            可以在 `google-services.json` 里找到. https://developers.google.com/android/guides/client-auth
