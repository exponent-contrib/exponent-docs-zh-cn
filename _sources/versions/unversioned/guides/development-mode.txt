.. _development-mode:

================
Development Mode
================

React Native 提供很多开发阶段非常有用的工具: Chrome 远程 JavaScript 调试, live reload,
host reloading, 还有元素审查类似 Chrome 里你经常使用的 inspector。在 app 运行期间也会做很多的检查，
比如你是否使用废弃属性，或者忘记给组件传递必要的属性。

.. figure:: img/development-mode.png
  :width: 100%
  :alt: Screenshots of development mode in action

**这些功能带来一些代价: 开发阶段你的 app 会相对慢一些。** 你可以在 XDE 里打开或者关闭 development mode。
当你切换的时候，为了生效请关掉然后重新打开你的app。 **任何时候你要测试 app 性能的话，请记得关掉 development mode。**.
