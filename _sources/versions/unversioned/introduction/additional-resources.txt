其他资源
====================

这里列举一些资源还有一些开源的项目来帮助你学习Exponent.

Exponent 文章
"""""""""""""""""
* `The Exponent Log <https://blog.getexponent.com/>`_ - 我们的官方博客
* `Building li.st for Android with Exponent and React Native <https://www.youtube.com/watch?v=cI9bDvDEsYE>`_ - 会议演讲
* `Build Cross Platform React Native Apps with Exponent and Redux <https://www.pluralsight.com/courses/build-react-native-exponent-redux-apps>`_ (PluralSight视频教程)

用Exponent开发的应用
""""""""""""""""""""""""""""""""""""
* `Growler Prowler <https://getexponent.com/@community/growler-prowler>`_ (`source <https://github.com/brentvatne/growler-prowler>`_) - 一个demo展示怎么使用maps, navigation, Facebook登录还有其他东西。
* `Floatyplane <https://getexponent.com/@exponent/floatyplane>`_ (`source <https://github.com/exponentjs/floatyplane>`_) - 一个你可能熟悉的游戏
* `Foodwheel <https://getexponent.com/@exponent/foodwheel>`_ (`source <https://github.com/exponentjs/foodwheel>`_) - 我们怎么决定去哪里吃饭
* `React Native Playground <http://rnplay.org>`_ (`source <https://github.com/exponentjs/rnplay>`_) - RNPlay应用
* `Instagram Stories Clone <https://getexponent.com/@mastermo/instagram-stories>`_ (`source <https://github.com/mastermoo/rn-instagram-stories>`_)
* `Pomodoro <https://getexponent.com/@exponent/pomodoro>`_ (`source <https://github.com/exponentjs/pomodoroexp>`_) - Pomodoro 应用

React Native
""""""""""""""

* `React Native Express <http://www.reactnativeexpress.com/>`_ - 逐步介绍React和React Native。
* `Official React Native Documentation <https://facebook.github.io/react-native/docs/sample-application-movies.html>`_
* `React Native Fundamentals <https://egghead.io/courses/react-native-fundamentals>`_ (Egghead.io视频教程)
* `Learning React Native <http://shop.oreilly.com/product/0636920041511.do>`_ (book)

React
"""""""""""""""

* `Official React Documentation <https://facebook.github.io/react/docs/getting-started.html>`_

JavaScript
"""""""""""""""""
